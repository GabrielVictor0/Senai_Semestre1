Motivação

HTML (abreviação de Hypertext Markup Language) é uma linguagem de
marcação utilizada n estruturação de páginas web. Sua sintaxe é bastante
simple, e assim como a XML é baseada em tags, que representam os
diversos elementos de uma página, como imagens e links.
